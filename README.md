# 浏览器版本检测

这是一个基于最新实验性 API `navigator.userAgentData` 的浏览器版本检测页面，使用 Vue3 和 NaiveUI 开发

[MDN 文档](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/userAgentData)

## 项目安装

```sh
npm install
```

### 开发运行

```sh
npm run dev
```

### 生产打包

```sh
npm run build
```

### [ESLint](https://eslint.org/)格式化

```sh
npm run lint
```
